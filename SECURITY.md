# Security Policy

## Reporting a vulnerability

Please **do not** report security issues in public GitHub issues.

Instead, report them privately via
[GitHub's private vulnerability reporting](https://github.com/Miragon/camunda-diagram-notion-embed/security/advisories/new),
or by email to **opensource@muenchendigital.io**.

Please include steps to reproduce and any relevant details. We will acknowledge your report and
keep you informed of the fix.
